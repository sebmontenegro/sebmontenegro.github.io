Examining Police Misconduct: Evidence from Chicago

I will study data from the Citizens Police Data Project which can be accessed via their website. The datasets can be downloaded at different grouping levels such as police districts, wards, beat, and neighborhoods. The data are presented in comma-separated value files that contain different variables describing the characteristics of the interaction between the police officer and the complaining witness.

You can access the data and code that I used to clean the data files here.

I will start by providing you with a couple of visualizations that are very interesting in order to understand whats going on within the Chicago PD.
